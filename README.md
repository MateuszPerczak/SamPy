# SamPy
SamPy allows you to find duplicated files on your hard drive.

## Python version:
3.8.2

## Used Libraries:

+ tkinter
+ typing
+ os
+ PIL
+ random
+ hashlib
+ threading
+ webbrowser

## Authors
Main programmer, designer: Mateusz Perczak
